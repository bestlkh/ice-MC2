    Authors

    All messages in the chat are sent by plain text. But there are many advanced message types that you may want to use.

    You can simply use Message class to auto generate the following text format. This is just for reference.

    Regular Text¶

    Simply send the message as plain text.

    Attachments¶

    Messages can have attachments.

    To attach extra data, simply append \n-----MC2 BEGIN ATTACHMENT-----\n to your message.

    Then append base64 encoding of the following JSON data.

    Then append \n-----MC2 END ATTACHMENT-----\n

    {
  "attachment-name": "data"
}

    For example:

    This is just a simple text message
-----MC2 BEGIN ATTACHMENT-----
eyJhdHRhY2htZW50LW5hbWUiOiAiZGF0YSJ9
-----MC2 END ATTACHMENT-----

    SVG Source¶

    With SVG source, user can put the message back to editor.

    {
  "svg-source": "SVG Source in base64, no mime"
}
