## How to Add a TA
1. Click the **TAs** menu item.
2. Click **Create TA**.
3. Enter the name of the **TA**.
4. Click **Submit**.

## How to Login in as a TA (in progress)
1. Enter the URL provided by the room owner.
2. Click **Login as TA**.
2. Enter a username and the secret.
