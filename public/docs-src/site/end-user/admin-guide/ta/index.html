    TA Management¶

    How to Add a TA¶

    1. Click TAs menu item
    2. Click Create TA
    3. Enter the name of the TA
    4. Click Submit

    How to Login in as a TA*¶

    1. Navigate to the namespace instance URL
    2. Click Login as TA
    3. Enter the secret.
